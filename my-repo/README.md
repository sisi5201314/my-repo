# My Git Demo Project
> 这是一个用于学习 Git 基础操作的示例项目

## 项目介绍
本项目主要用于练习 Git 的核心操作，包括：
- 仓库初始化与提交
- 分支创建、切换与合并
- 提交记录查看与版本回退
- 文件状态管理

## 环境要求
| 软件       | 最低版本 | 下载地址                          |
|------------|----------|-----------------------------------|
| Git        | 2.20.0   | [Git 官网](https://git-scm.com/)  |
| VS Code    | 1.80.0   | [VS Code 官网](https://code.visualstudio.com/) |
